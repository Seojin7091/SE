### Use Case Description: 설문 응답

| Actor Action | System Response |
|---|---|
| | 1. 설문 상세 정보 조회 화면에서 현재 진행 중인 설문에 한해 응답 버튼을 활성화하여 표시한다. |
| 2. 응답 버튼을 누른다. | |
| | 3. 응답 화면으로 이동하여 설문 문항과 응답 항목을 표시한다. |
| 4. 설문에 응답한다. | |
| 5. 응답 버튼을 눌러 완료한다. | |
| | 6. 응답 완료 메시지를 표시한다. |

Alternative Courses
Step 1: 진행 중이 아닌 설문인 경우 응답 버튼이 비활성화되어 응답할 수 없다.

---

### Use Case Description: 응답한 설문 조회

| Actor Action | System Response |
|---|---|
| | 1. 회원이 응답한 모든 설문 목록을 한 화면에 표시한다. 각 항목에는 설문 제목, 응답 내용, 응답 일시가 출력된다. |

Alternative Courses
Step 1: 응답한 설문이 없는 경우 빈 목록을 표시한다.

Extensions
After step 1: 회원은 현재 진행 중인 설문에 한해 취소 버튼을 눌러 응답을 취소할 수 있다.

---

### Use Case Description: 응답한 설문 수정

| Actor Action | System Response |
|---|---|
| | 1. 응답한 설문 조회 화면에서 현재 진행 중인 설문에 한해 수정 버튼을 활성화하여 표시한다. |
| 2. 수정할 설문의 수정 버튼을 누른다. | |
| | 3. 응답 수정 화면으로 이동하여 기존 응답 내용과 가장 최근 수정 날짜를 표시한다. |
| 4. 응답 항목을 수정한다. | |
| 5. 수정 버튼을 눌러 완료한다. | |
| | 6. 수정 완료 메시지를 표시한다. |

Alternative Courses
Step 1: 종료된 설문인 경우 수정 버튼이 비활성화되어 수정할 수 없다.
